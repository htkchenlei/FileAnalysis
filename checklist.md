## 二、UI/UX 检查

### 2.1 颜色对比度 (WCAG AA)
- [ ] 主文字 (#4A4A4A) 在背景 (#FFF9F0) 上对比度 ≥ 4.5:1
- [ ] 次要文字 (#8A8A8A) 在背景上对比度 ≥ 4.5:1
- [ ] 按钮文字 (#FFFFFF) 在主色背景 (#B8A9C9) 上对比度 ≥ 4.5:1
- [ ] 成功/警告/错误状态颜色可区分

### 2.2 触摸目标尺寸
- [ ] 按钮最小尺寸 44×44px (iOS) / 48×48dp (Android)
- [ ] 上传区域足够大
- [ ] 列表项可点击区域足够
- [ ] 导航项可点击区域足够

### 2.3 间距系统
- [ ] 基础间距单位 4px
- [ ] 组件内边距 24px
- [ ] 页面边距 24px (移动端) / 48px (桌面端)
- [ ] 元素间距一致

### 2.4 动画效果
- [ ] 过渡动画时长 150-300ms
- [ ] 使用 ease-out 缓动函数
- [ ] 悬浮状态有视觉反馈
- [ ] 点击状态有视觉反馈
- [ ] 加载动画流畅

### 2.5 响应式布局
- [ ] 移动端布局正常 (< 768px)
- [ ] 平板布局正常 (768px - 1024px)
- [ ] 桌面布局正常 (> 1024px)
- [ ] 无水平滚动条
- [ ] 内容宽度合理 (最大 1280px)
